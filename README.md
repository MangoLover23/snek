## Gebruiken als extensie

Deze repository kan worden toegevoegd als **extensie** in MakeCode.

* open [https://makecode.microbit.org/](https://makecode.microbit.org/)
* klik op **Nieuw project**
* klik op **Extensies** onder het tandwielmenu
* zoeken naar **https://github.com/mangolover23/snek-game** en importeren

## Dit project bewerken ![Badge buildstatus](https://github.com/mangolover23/snek-game/workflows/MakeCode/badge.svg)

Om deze repository te bewerken in MakeCode.

* klik op **Importeren** en klik vervolgens op **Importeer URL**
* plak **https://github.com/mangolover23/snek-game** en klik op importeren
